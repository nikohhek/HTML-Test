Einführung

HTML & CSS verstehen

Eine einfache Präsentation für den Einstieg: Was HTML macht, was CSS macht und wie daraus eine kleine, realistische Website entstehen kann.

Ohne JavaScript. Klar. Einfach. Direkt.

Was ist HTML?

HTML ist die Struktur einer Website. Es sagt dem Browser: „Das hier ist eine Überschrift“, „das ist ein Text“ oder „das ist ein Bild“.

Überschrift

Damit die Seite sofort verständlich wirkt.

Absatz

Für normale Texte und Erklärungen.

Link

Um zu einer anderen Seite oder Folie zu springen.

Was ist CSS?

CSS macht die Seite schön. Es bestimmt Farben, Abstände, Schriftarten und das gesamte Layout. Ohne CSS wäre eine Website oft nur eine einfache Textliste.

Mit HTML

Die Seite hat Inhalt und Sinn.

Mit CSS

Die Seite sieht ordentlich und modern aus.

Wie arbeiten beide zusammen?

HTML baut das Gerüst. CSS gestaltet das Gerüst. Man kann sich das wie ein Zimmer vorstellen: HTML sind Wände, Tisch und Stuhl. CSS sind Farbe, Deko und Licht.

HTML = Inhalt und Struktur

CSS = Aussehen und Gestaltung
